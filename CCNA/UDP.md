#NetworkFundamentals #ccna-day30

### Overview
Connectionless protocol.

Does not provide:
- reliable communication
- sequencing
- flow control

Faster than TCP at the cost of additional overhead.
Preferred for real-time applications like video or voice.

### Port Numbers
- DHCP server (67)
- DHCP client (68)
- TFTP (69)
- SNMP agent (161)
- SNMP manager (162)
- Syslog (514)
- DNS (53)
