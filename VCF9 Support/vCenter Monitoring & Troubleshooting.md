### Service Management
```
service-control --list: Lists vCenter Appliance services.
service-control --status: Lists vCenter Appliance services status.
service-control --all --start: Start all vCenter Appliance services.
```

### VMware Appliance Management Interface (VAMI)
You can use the VAMI to perform a file based backup, allowing you to back up:
- Configuration
- Inventory
- Historical Data

The backup can be streamed to a remote repository using:
- FTP
- HTTPS
- SCP
- NFS
- SMB
