#vcf9 

### Overview
By embetting INT instructions into the Geneve header option field, the feautre offers the following abilities:
	- Tracing live packets.
	- Capturing packets.

Live Traffic Analysis uses a flow analysis framework that uses In-band Network Telemtry (INT) to collect per-packet and per-hop monitoring information.

### Configuration
To configure a traffic analysis session between VMs, navigate to **Plan and Troubleshoot > Troubleshooting tools > Traffic Analysis** and click **New Session**.

The resulting Live Traffic Analysis includes the logical network topology between the source and the destination. PCAP capture files are available for download for further detailed analysis.
