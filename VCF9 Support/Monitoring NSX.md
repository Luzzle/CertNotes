### Overview
NSX can be monitored from VCF Operations by navigating to **Infrastructure Operations > Network Operations**.
For additional insight, you should install the Operations Network Appliance.

##### Operations Network Appliance
Operations Network Appliance provides visibility across the VCF networking stack. This includes 2 different appliances:
	- Platform VM processes network data.
	- Collector VM collects the data from the VCF environment and sends it to the platform VM for processing.

##### Analyze Flows
The Analyze Flows tab displays the busiest applications in your environment classified by flow volume, traffic rate, session count and flow count.

##### Network Performance
The Network Performance tab display normal and abnormal flows based on their TCP Round Trip Time (RTT).
